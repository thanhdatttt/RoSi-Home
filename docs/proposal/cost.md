# Estimated Cost — RosiHome
 
| Category | Service / Upgrade Detail | Estimated Cost (VND) | Role & Practical Effect |
|---|---|---|---|
| **Development Tools** | Claude Code / Claude Pro upgraded plan, shared across the 5-person dev team | 1,600,000 đ (~2 months) | Lets the dev team generate code, test with the billing engine, QR-payment logic, and mobile app scaffolding faster, and auto-fixes bugs during development |
| **AI Agent Infrastructure (LLM APIs)** | Pay-as-you-go API credit (Claude Sonnet or GPT-4o) for in-app AI features | 900,000 đ (pay-as-you-go) | Powers: AI reading of utility-meter photos to auto-fill electricity/water readings, and an AI-generated weekly summary of revenue/occupancy/arrears on the landlord dashboard. Enough for ~400–600 smart-processing calls during the pilot |
| **Cloud Infrastructure** | Upgraded cloud server + database (small production tier) | 800,000 đ (~2 months) | Prevents timeouts/lag when the AI features (meter-reading recognition, report generation) need to process data or images |
| **Domain & Security** | 1 domain (.com or .vn) + SSL certificate | 450,000 đ | Builds trust for landlords and tenants accessing the app; protects tenant and lease data |
| **System Contingency** | Buffer for AI API overage / rate-limit spikes during testing | 500,000 đ | Covers the dev team's testing and experimentation with AI features that consume extra tokens |
| **TOTAL** | Cost to build & pilot the AI-assisted MVP | **4,250,000 đ** | ~4.25 million VND for the entire project |
 
## Notes
 
- **No labor cost:** unlike a commercial project, developer time is not counted — it's a coursework contribution, not paid work. Every number above is cash the team would actually spend.
- **AI is used in two ways**, both reflected above: (1) as a *development tool* (Claude Code/Claude Pro helping the team write code faster) and (2) as a *product feature* (in-app AI reading meter photos and summarizing reports for the landlord) — the "Development Tools" and "AI Agent Infrastructure" rows are deliberately separate line items because they're different costs for different purposes.
